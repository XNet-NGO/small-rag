PROJECT LOCATION: /home/user-x/projects/small-rag/

WHAT IS SMALL-RAG?
  A self-contained, portable RAG system for autonomous agents (like AX)
  - Single Go binary (30MB)
  - Zero runtime dependencies
  - Local-first (no cloud)
  - Agent-first API (REST)
  - Fully portable data

CORE STACK
  Language:      Go 1.25
  Vector Store:  SQLite (pure-Go driver)
  Embeddings:    Qwen3-Embedding-0.6B (CPU)
  HTTP:          Chi router
  Streaming:     Server-Sent Events (SSE)

USAGE WITH AX
  Terminal 1: ./small-rag
  Terminal 2: ./ax -p "Search knowledge base for ML concepts"
  
  AX automatically calls rag_search tool
  Results included in LLM context
  LLM generates grounded answer
